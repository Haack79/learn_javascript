Javascript has dynamic typing - meaning data types are automatically assigned to variables so you don't have to manually assign the type. number - floating point for decimals and integers , so even if you see 5 it is 5.0 string - sequence of characters used for text Boolean logical true or false Undefined - data type of a variable that does not have a value yet.
